【HTML(Hypertext Markup Language)】


1.html 언어는 웹 사이트 제작시 가장 기본(기초가 되는 프로그램 언어라고 볼 수 있는데
      -이 언어를 이용하여 웹 문서를 제작할 수 있고,
      -이동 명령을 통해 다른 웹 문서로의 이동이 가능하며 여러 명령어들을 이용하여 단락, 목록, 표 등의 표시도 가능하다.
2.Html 언어의 코딩은 메모장으로도 만들 수 있지만 일반적으로 간편한 웹 편집기를 많이 사용한다.
      *편집기는 여러가지가 있지만 아래의 편집기들을 일반적으로 많이 사용한다.
        -에디트플러스(Editplus),    울트라에디트(ultra)
        -서브프임텍스트(SublimeText),  아톰(Atom)
      *각 툴들을 모두 섭렵할 필요는 없고 서로들 유사한 부분이 많아 어느것 하나 익숙하게 되면 다른 툴 사용 시 특별한 어려움은 없다.
3.Html은 언어 형식에 맞도록 태그를 작성하여 작업 후 저장 시 반드시 *.html(혹은 *.htm)이라는 확장자로 저장을 한다.

★html 언어는 브라우저에서 해석이 되는 언어이다.
  해석되는 속도는 굉장히 빠르나 브라우저 마다 해석되어 표현되는 결과가 모두 같은 것이 아니라서
  Html을 학습 단계에서는 잘 못느끼지만 실무 적용 단계에서는 큰 고민일 수 밖에 없음.

【다운로드 및 설치】 1.우리 Html 실습에서는 에디트플러스(editplus) 툴을 기본 툴로 설치하여 사용하도록 하겠다. 2.구글에서 영문 또는 한글로 (editplus 또는 에디트플러스) 검색 후 해당 사이트로 이동하여 평가판(1개월 무료)을 다운받아 설치한다.
※설치는 강사와 같이 진행한다.
【기본구성】
1.웹 페이지의 제목은 태그에 작성을 하고 내용은 태그에 작성을 한다.
2.HTML의 모든 명령어는 태그(tag)라는 공툥명을 뒤에 붙여 호명한다.
3. HTML은 대소문자를 구별하지 않는다.
<html> <head> <title>제목작성</title> </head> <body> ::내용작성:: </body> </html> 3.저장 시에는 반드시 확장명을 html로 하여 저장한다.

▣.기본 구성 태그
 1.html:웹 페이지 작성을 할 때 맨 밖에 형성을 하여 html 문서를 알리는 역할을 한다.
 2.head:웹 페이지의 제목표기/다른 페이지와의 스타일 연결/ 기본 인코딩 세팅등을 할 수있는 공간을 제공한다.
 3.title:웹 페이지의 실제 제목을 표기한다.
 4.body:웹 페이지에서 출력을 담당한다.

(예제)
<html> <head> <title>첫번째 예제</title> </head> <body> 자바기반 풀 스택 웹 개발자 과정으로 HTML, JAVA등을 학습합니다. </body> </html>

c-1.<font>태그
 - 글자의 크기/ 서체 / 색상을 표현할 수 있는 태그
  § 아쉽게 지금은 사장된 태그이나 HTML 폰트 관련 기본 이해를 하는데 도움이 되므로 학습을 한다.
 (속성)
 -size (1-7), face (굴림,돋움,궁서), color(영문 및 RGB 색상)
 (예제)
 <font size="1">javascript</font><br>
 <font size="2">javascript</font><br>
 <font size="3">javascript</font><br>
 <font size="7">javascript</font><br>
 <font size="10">javascript</font><br>
 <font face="궁서체">오라클</font><br>
 <font color="blue">html</font><br>
 <font color="#cc0099">project</font><br>
 <font size="5" FACE="돋움체" color="skyblue">


c-2 <H>태그
 -태그 뒤에 숫자를 붙여 사이즈를 결정한다.
